  • list-resource-loader-paths - Provides a list of filesystem paths of the module, if there are any and the module loader exposes them.
    Deprecated Since 1.4.0
    Experimental and unsupported.
    Request Parameter Type Required Expressions Allowed Default value Description
    module STRING true false The identifier of the module, in [:] format. If the slot is not provided, a value of "main" is assumed.

    Reply properties

    A list, each of whose elements is a filesystem location from which the module can load resources.

    LIST
    Value Type
