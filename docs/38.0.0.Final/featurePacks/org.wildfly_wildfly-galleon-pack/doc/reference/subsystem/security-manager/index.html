Management API Reference for subsystem=security-manager

The Security Manager subsystem

Deprecated Since 4.0.0
The use of the Java security manager is being disabled in Java under JEP-486, additionally security manager integration is not supported with Jakarta EE 11. The security manager extension will be removed in a future version.
